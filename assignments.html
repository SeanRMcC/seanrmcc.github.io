Assignment 0
This simple script will take in two numbers from the user and print them to the command line.
Assignment 1
Data collection and analysis: In this assignment, the whole class collected training data on the arrivals of the 6 elevators in CDS. After this, I had to visualize and determine the optimal place to stand. Then, the class collected test data to test our best place to stand.
Assignment 2
KMeans Visualization: In this assignment, I implemented the KMeans algorithm from scratch in React.js. I then used Plotly.js to visualize the algorithm and allow users to modify k, the initialization method of the centers, and other aspects of the algorithm.
Assignment 3
Custom SVD & MNIST Logistic Regression: In this assignment, I implemented singular value decomposition using only numpy functions. With this algorithm, I preprocessed the MNIST data set and used such data to train a logistic regression model, using a different number of SVD components.
Assignment 4
LSA Search Engine Webpage: Using latent semantic analysis, I vectorized (using tfidf) to corpus of documents and when a search query was provided, I computed the top 5 similar documents (using cosine similarity) and returned said documents and visualized them.
Assignment 5
K-Nearest Neighbors Kaggle Competition: I developed a KNN classifier from scratch to predict the likelihood of a member of the bank leaving said bank. To pre-process the data, I used one hot encoding, scaled the features between 0 and 1, and reduced the dimensionality using SVD. I then performed cross validation to prevent overfitting while selecting the best hyper parameters.
Midterm Competition
Amazon Movie Reviews Kaggle Competition: For the midterm, we had to train a model that would predict the number of stars that a review for a movie. I ended up extracting both row based and aggregated statistical features, using said features to train a HistGradientBoostingClassifier locally.
